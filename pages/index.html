Constraint Satisfaction Problems

Chapter 6

State = factored representation (set of variables with value). Constraint Satisfaction Problem = is a problem defines by variables and constraints, the problem is solved if all variables have a value and all the constraints are satisfied.

Components of CSP:

  • X is a set of variables
  • D is a set of domains one for each variables
  • C is a set of constraints that specified allowable combinations of values

A state in CSP is an assignment of values to some or all of the variables. Consistent assignment if does not violate any constraints, complete assignment if all variables are assigned, partial assignment if some variables are assigned, solution if the assignment is complete and consistent.

D3.js functions summary

d3.select("#x") = select the DOM node with id equal to "x"

d3.selectAll(".xs") = select DOM nodes with class equal to "xs"

domNode.style("p", "v") = add CSS style attribute with property "p" with value "v" to the domNode

domNode.attr("p", "v") = add attribute "p" with value "v" to domNode

domNode.classed("c", bool) = append class "c" if true or remove it

domNode.on("event", function) = give interactivity to domNode

domNode.transition().otherFun() = transition from prev state to new state determine by otherFun()

  • .delay(millisec)
  • .duration(millisec)

domNode.html(v) = insert inner value into domNode

d3.csv("fileName.csv",function(error,data) { ... }) = import data from a csv file

d3.scaleLinear().domain([a, d]).range([b, c]) = scale the value from domain (large) to range (smaller)

d3.scaleLinear().nice() = round scale points for human

d3.scaleLinear().rangeRound() = in place of range, it will scale in the rounded range

d3.scaleLinear().clamp(true) = excessive values will be rounded to the range’s low or high value

d3.max(dataset, function(d)) = found the max value from the returned value of function(d)

d3.min(dataset, function(d)) = same as top but now is minimun value

d3.range(l) = quickly generate an array of sequential numbers

domNodes.data(data) = bind data with domNodes selected

domNodes.data(data).enter() = creates new domNode if data exceed the length of domNodes

  • append(n) = add a new DOM node define in n
  • insert(n)
  • attr("a", "v")

domNodes.data(data).exit() = ignores data that exceed the domNodes, the nodes without corresponding nodes are removed

  • remove() = remove DOM node with no data associated

d3.axisBottom(scale) = create an axis with ticks label on bottom, scale define the scale of axis

axisBottom & axisTop = are horizontal axes

axisTop & axisBottom = are vertical axes

axis.scale(x) = set scale x to axis

axis.ticks(n) = set a desirable min value of ticks on axis

axis.tickValues([t1, t2, ...]) = set manually the ticks on axis

nodesToUpdate.transition().YOUR_UPDATE = generate transtion from previous params to new params

.transition().duration(n) = set a duration n in milliseconds of the transition

.transition().ease(easeMode) = set easeMode as ease transtion:

  • d3.easeLinear
  • d3.easeCircleIn
  • d3.easeElasticOut
  • d3.easeBounceOut
  • More here

.transition().on("start" || "end", fun) = execute fun on start/end of transition

<svg>

Your canvas in which everything is drawn.
  • Top-left corner is 0.0
  • x-axis is drawn left-to-right, the y-axis is drawn top-to-bottom

Elements inside:

  • circle & ellipse
    • cx and cy determine the center of the circle
    • r determines the radius of the circle
  • rect
    • x and y: determine the shape’s top-left corner
    • height and width: that determine its overall form
  • line
  • x1 and y1 determine the starting point
  • x2 and y2 determine the end point
  • polygon
  • text
  • g: logical grouping of elements
  • path
    • d:
      • Z at end determines the close of path (close shape)

In order to style these elements use:

  • fill
  • stroke, stroke-width, stroke-dasharray

Move these elements? Add attribute transform equal to:

  • translate(x, y)
  • scale(v)
