Policy Iteration

  1. Start with an arbitrary policy π0
  2. Evaluate the utilities given that policy: Ut(s) = R(s) + γ ∑s' T(s, πt(s), s') Ut(s')
  3. Improve the policy selected: πt+1 = argmaxa ∑s' T(s,a,s') Ut(s')
  4. Iterate: step 2

Now the bellman equation have no max operation in it. It is a linear system.

Bellman equations

Value

V(s) = maxa( R(s, a) + γ ∑s' T(s, a, s') V(s') )

V(s') is the value from the next state taken the action a.

Quality

Q(s, a) = R(s, a) + γ ∑s' T(s, a, s') maxq'Q(s', q')

Continuation

C(s, a) = γ ∑s'T(s, a, s') maxa' ( R(s', a') + C(s', a') )
